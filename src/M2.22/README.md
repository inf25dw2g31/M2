# Projeto Web 2º Ano - React

Projeto inicial em React usando Vite para HTML, CSS e componentes. Ideal para trabalhos de Web no 2.º ano do curso de informática.

## Comandos

- `npm install` - instala as dependências
- `npm run dev` - inicia o servidor de desenvolvimento
- `npm run build` - cria a versão de produção
- `npm run preview` - pré-visualiza o build de produção

## Estrutura

- `index.html` - ponto de entrada HTML
- `src/main.jsx` - bootstrap do React
- `src/App.jsx` - componente principal com rotas
- `src/index.css` - estilos globais
- `src/App.css` - estilos principais da aplicação
- `src/components/NavBar.jsx` - navegação entre páginas
- `src/components/Footer.jsx` - rodapé do site
- `src/pages/Home.jsx` - página inicial
- `src/pages/Sobre.jsx` - página sobre
- `src/pages/Projetos.jsx` - página de projetos com lista dinâmica
- `src/pages/Contato.jsx` - página de contato com formulário
- `src/data/projetos.js` - dados de exemplo para a página de projetos

## React Router

Este projeto usa `react-router-dom` para navegação de páginas. Após instalar as dependências, use `npm run dev` e acesse `http://localhost:5173`.
