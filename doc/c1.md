# C1 : FestivAll — Aplicação Web Cliente em ReactJS

## Description

O FestivAll é uma Aplicação Web Cliente desenvolvida em ReactJS, no âmbito do segundo momento (M2) de avaliação contínua da unidade curricular de Web Development 2. A aplicação consome a **Ticketmaster Discovery API**, uma API REST pública protegida por Consumer Key, permitindo ao utilizador explorar eventos culturais e desportivos em tempo real — concertos, festivais, peças de teatro e eventos desportivos — em várias cidades e países.

A aplicação foi construída com **React 18** e **Vite** como ferramenta de build, recorrendo a **Material UI (MUI)** para os componentes de interface, **Framer Motion** para animações, **React Router v6** para a navegação e **Context API** para gestão de estado global. Toda a aplicação é containerizada com **Docker** (build multistage Node + Nginx) e publicada via **GitHub** e **DockerHub**.

A camada de autenticação é gerida no cliente através de um contexto (`AuthContext`) que valida a presença de uma Consumer Key da Ticketmaster antes de permitir o acesso às rotas protegidas. Sem chave válida, o utilizador é automaticamente redirecionado para a página de login pelo componente `RequireAuth`.

Funcionalidades principais:
- Login com Consumer Key da Ticketmaster
- Dashboard com eventos em destaque
- Pesquisa de eventos com filtros (palavra-chave, cidade, data, categoria)
- Detalhe de cada evento (data, local, artistas, preços, link para bilheteira)
- Pesquisa de artistas (attractions)
- Sistema de favoritos com persistência no `localStorage`
- Interface responsiva (mobile + desktop)
